# Project files

This kit works best when each project repository has a small set of project-specific files.

## AGENTS.md

Required or strongly recommended.

Use it for:

- branch rules
- test commands
- PR rules
- forbidden operations
- security notes
- project-specific workflows

## docs/current-status.md

Use it for:

- current state
- recent completed work
- known issues
- next steps

## docs/TODO.md

Use it for:

- pending tasks
- priority order
- follow-up candidates

## Principle

Keep common Codex operation rules in this repository.
Keep project-specific rules in each project repository.
