# FAQ

## Do I need one setup per project?

No. Install the AutoHotkey script once on a Windows PC and use the hotstrings in any Codex project.

## Do I need setup on another PC?

Yes. This is local Windows input automation, so each PC needs its own setup.

## Where do project-specific rules go?

In each project's `AGENTS.md` and docs.

## Should secrets be placed in this repository?

No.
